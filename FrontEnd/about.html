Members:
Stephan Heijl
Thierry Haddad
Tycho Canter Cremers
Roel Beumers


This application was made in order of HAN BioCentre.
As described on the homepage, the web-application contains a large database
with general information and links to the relevant articles, corresponding to
the subject of the users' input. With the aid of this tool, searching PubMed's
articles for a combination of keywords becomes incredibly more efficient. In addition to searching for new keywords, it's also possible to navigate
trough the existing data, stored in a SQL database.


General technical information:
The used software is including, but not limited to, the following:
- Python version 2.7
- Notepad++ version 6.3.3
